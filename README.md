# Recipe Box

Este projeto foi feito para o estudo de Ruby on Rails, Ã© baseado no tutorial do [Mackenzie Child - Week 3: How To Build A Recipe Box With Rails 4](http://mackenziechild.me/12-in-12/3/), o projeto tem como finalidade criar um lista para receitas, utilizando imagens e gems muito conhecidas do mercado como:

* [Devise](https://github.com/plataformatec/devise)
* [Bootstrap-sass](https://github.com/twbs/bootstrap-sass)
* [Cocoon](https://github.com/nathanvda/cocoon)
* [PaperClip](https://github.com/thoughtbot/paperclip)
* [Haml](http://haml.info/)
